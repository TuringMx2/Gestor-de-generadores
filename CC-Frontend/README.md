# CC Frontend

Frontend inicial de CC construido con Next.js, TypeScript y App Router.

## Requisitos

- Node.js 20+

## Instalacion

```bash
npm install
```

## Ejecucion local

```bash
npm run dev
```

## Estructura principal

- `src/app`: rutas de la aplicacion.
- `src/features`: modulos funcionales.
- `src/shared`: codigo reutilizable.
